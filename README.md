# hearth-lore

Статический сайт лора Лунны для GitHub Pages.

## Что важно

- Главная страница: `index.html`.
- Чистые адреса сделаны через папки с `index.html`: `test/`, `crosswords/`, `pages/lunna/` и так далее.
- На GitHub Pages страницы будут открываться без суффикса `.html`.
- Для локальной проверки чистых адресов лучше запускать папку через локальный сервер, например: `python -m http.server`.
- `.git` в архив не включён.
