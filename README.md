# Тайная библиотека Лунны

Лор-сайт Лунны, Реми, Очага и интерактивной игровой комнаты.

## Структура

- `index.html` — главная страница с лорным входом в дом Лороведьмы
- `test.html` — страница под тест “стать частью Очага”
- `crosswords.html` — страница игровой комнаты с кроссвордами
- `pages/` — вики-страницы обитателей дома
- `assets/portraits/` — портреты персонажей
- `style.css` — оформление
- `script.js` — заготовка для интерактива

## GitHub Pages

`Settings → Pages → Deploy from branch → main → /root`
