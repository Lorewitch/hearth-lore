# Отчёт по версии с листами

Что изменено:

- Главная страница снова основана на основном дизайне, без экспериментальной книги.
- Главная разбита на три полноэкранных листа со scroll-snap:
  1. приветствие в «Мягком корешке»;
  2. чертежи здания;
  3. обитатели дома.
- Из верхнего меню главной убраны пункты «Главная», «Чертежи», «Обитатели».
- Для трёх листов добавлена боковая тонкая нумерация 01 / 02 / 03.
- На внутренних страницах прямые ссылки на «Мягкий корешок», «Чертежи» и «Обитателей» сохранены в верхнем меню.
- Карточки обитателей на главной больше не карусель: все шесть видны сразу сеткой.
- Исправлено закрытие модалки чертежа: больше не используется `body position: fixed`, из-за которого страница могла прыгать вверх и возвращаться к секции.
- `.git` из архива удалён, чтобы не загружать служебную историю репозитория.

Проверки:

- `script.js` проходит синтаксическую проверку Node.js.
- Проверены локальные ссылки на HTML, CSS, JS и изображения.
- Пустых `href="#"` нет.
